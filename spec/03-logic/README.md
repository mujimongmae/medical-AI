- **핵심 판단 규칙:** 입력 → 처리 → 출력의 분기·엣지 케이스
- **AI 프롬프트:** Claude API 시스템/유저 프롬프트, RAG 소스, 출력 스키마
- **평가/검증:** 로직 정확성·안전성 확인 방법
