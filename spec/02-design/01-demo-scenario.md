## 1. 목적
**"강원 독거노인 마을 공동체 응급대응"** 을 한 편으로 시연: 쓰러짐 감지(모의) → 환자 폰 **15초 사이렌** → 무반응 시 **119 자동신고(모의) + 화면 꺼진 이웃 폰을 푸시로 깨움** → 이웃이 **단일 질문 트리아지 → CPR 자동 진행 / 또는 AI 음성 추천** 수행.

> ⚠️ 전제: 영상인식·119·소방 연계는 **모의**. 데이터 **합성만**. 응급 **119 최우선**, 확정 진단·처방 없음, 디스클레이머 상시.

## 2. 폰 배치 (팀 보유: 갤럭시 1 + 아이폰 1)
| 폰 | 역할 | 준비 | 푸시 |
|---|---|---|---|
| **아이폰** | **환자** (앱 켜둠·화면 켜둠) | **Safari로 터널 URL 접속** → "어르신(환자)" **탭**(=오디오 무장) → 병력 입력(예: 고혈압, 협심증) | ❌ 불필요 (앱 켜둠 → WebSocket로 즉시 사이렌) |
| **갤럭시** | **이웃** (등록 후 화면 끔) | **apk 설치** → "이웃" 선택 → **알림 권한 허용**(토큰 등록) → **화면 끄기** | ✅ **FCM 필수** |
| 맥북 | 브로커 | `npm run dev:all` + `cloudflared tunnel --url http://localhost:5173`. 콘솔 로그 = 보조 화면 | — |

- **왜 아이폰=Safari?** 환자는 앱을 켜두는 역할이라 **푸시가 불필요** → iOS 푸시(유료 APNs) 우회. Safari로 터널 URL만 열면 설치 없이 동작. (원하면 네이티브 iOS 앱도 가능하나 데모 필수 아님)
- **왜 갤럭시=이웃?** 화면 꺼진 상태에서 깨우려면 **FCM 푸시**가 필수인데 Android(FCM)가 무료로 됨.
- ⚠️ **아이폰은 반드시 역할을 "탭"** 해야 오디오가 무장됨(iOS 자동재생 제약). 화면 켜두고 앱 포그라운드 유지(잠그면 사이렌 안 옴).

**트리거(쓰러짐 발생):** 아이폰 환자 화면의 `[데모]` 버튼, 또는 PC 브라우저 **`localhost:8787/admin`** 에서 해당 환자 "응급 발생".

**Claude(선택):** 맥북 `.env.local`에 `ANTHROPIC_API_KEY` 있으면 AI 추천이 실제 Claude, 없으면 폴백. 키는 서버 전용.

## 3. 등장 인물 (합성)
- 시드(`server/seed.ts`): 마을 `방림리`에 환자 **김복순(합성, 고혈압·협심증)** + 이웃 4명. 등록/병력은 `server/data.json`에 영속.
- 폰으로 등록하는 사용자도 `방림리` 소속 → 이웃 매칭 대상.

## 4. 시연 순서 (약 4~5분)

### 0) 도입 (30초)
- "강원은 병원이 멀어 골든타임을 놓칩니다. **발견 지연 + 초기 대처 공백**을 마을 공동체로 메웁니다." (근거: 농촌 OHCA 생존 3.4% vs 도시 8.7%, 강원 급성심정지 7.2% 전국 최저 — `screening/`)

### 1) 등록 (40초)
- **아이폰(환자)**: Safari 터널 URL → 이름·병력 입력 → **"어르신(환자)" 탭** → "● 정상 감시 중".
- **갤럭시(이웃)**: 앱 실행 → 이름 → "이웃" → **알림 권한 허용** → "● 대기 중" → **화면 끄기**.
- 멘트: "설치는 하나, 역할만 다릅니다."

### 2) 쓰러짐 → 15초 사이렌 (40초)
- 아이폰 `[데모] 쓰러짐 발생` (또는 PC `/admin`에서 트리거). 실서비스에선 영상인식 `POST /fall-event`가 대체.
- 아이폰: **전체화면 빨강 + 초대형 카운트다운 + 사이렌(두 음 왕복)·진동.**
- "오탐이면 '괜찮아요' 탭 → 즉시 해제." 이번엔 **누르지 않고 15초 무반응**.

### 3) 119 모의 + 화면 꺼진 이웃 깨우기 (30초) ★핵심
- 0초 → 아이폰 "도움을 요청했습니다"(초록).
- **맥북 콘솔**: `[MOCK 119] 자동 신고` 가 이웃 호출보다 **먼저** → 이어 `이웃 호출 N명`.
- **화면 꺼져 있던 갤럭시가 FCM 푸시로 깨어남** → 소리+알림 → 탭 → 앱 진입. (이 장면이 데모 하이라이트)

### 4) 단일 질문 트리아지 (20초)
- 갤럭시: "도착했습니다" → **"환자가 반응이 있고, 숨도 정상적으로 쉬나요?"**
  - **"아니요(반응 없음/숨 안 쉼)"** → CPR 화면 / **"예"** → AI 음성 추천.
- 멘트: "생명 직결(CPR 여부)은 **LLM이 아니라 규칙**이 판정."

### 5-A) CPR 자동 진행 (50초)
- **5초 카운트다운 → 자동 시작**(버튼 누를 필요 없음).
- **좌우 2분할:** 좌=압박부위(가슴 정중앙 O표시) 사진 / 우=전문가 흉부압박 시범 영상(반복). 미디어는 `app/public/cpr/`(`site.jpg`·`compressions.mp4`)에 넣고 재빌드 → 없으면 그림/애니메이션 폴백. ⚠️ **압박부위=가슴 정중앙(젖꼭지 사이, 흉골 하부). 명치(칼돌기) 금기.**
- **압박 1회마다 '딸깍' 클릭음 + 진동**(분당 110회 메트로놈).
- **30회 자동 → 인공호흡 화면 전환**: 별도 그림(머리 젖히고 숨 불어넣기) + **"1번 부세요 → 2번 부세요"** 단계 카운트 → **다음 30회 자동 재개**(반복). 수동은 중지만.
- 재개 시 압박 카운트는 **1부터 정확히**(이전 2-1-2 튐 버그 수정). 큰 문구 + 디스클레이머 상시.

### 5-B) AI 음성 추천 (의식/호흡 있는 경우, 50초)
- 진입 즉시 **마이크 권한 요청 + 자동 녹음**. 듣기 시작 시 **가벼운 신호음 + "🎤 듣고 있어요. 말씀하세요"**(초록·맥박) → 말을 멈추고 ~2.5초 지나면 **완료음 + "✅ 잘 들었어요"**(파랑)로 전환(자동으로 인식 종료). "버튼 누르고 말해야 하나?" 혼동 제거.
- "피가 많이 나요" 말함 → **완료 — AI 추천 받기**.
- 서버가 **병력 + 증상**을 Claude로 종합 → **"○○ 가능성이 높아요" + "지금 이렇게 하세요"** 큰 글씨 → "자세한 방법 보기".

### 6) 마무리 (20초)
- "발견→신고→초기대처 공백을 **마을 이웃 + 규칙 트리아지 + AI 추천**으로. 데모는 맥북 브로커지만 **Supabase/Vercel로 스왑** 가능."

## 5. 푸시 테스트 요령 (관제 페이지)
- PC 브라우저 **`localhost:8787/admin`** → 등록 사용자 목록 + "응급 발생" 버튼.
- **갤럭시 이웃 푸시 확인**: 갤럭시 이웃 등록(토큰) → 화면 끔 → `/admin`에서 **"시드 환자로 즉시 이웃 호출"** → 잠긴 갤럭시에 **약 25초 경보음**(알람 채널 `emergency_siren`) + 알림 → 탭하면 앱 진입, **호출 화면에서도 인앱 사이렌** 지속(“도착했습니다” 누를 때까지).
- **아이폰 환자 15초 확인**: 아이폰 환자 등록 → `/admin`에서 그 환자 "응급 발생" → 아이폰 사이렌(앱 켜둔 상태).

### 알림 동작 규칙 (중복·경보음)
- **1기기=1수신자**: 앱이 토큰 등록(`/api/push-token`) 시 같은 토큰을 물고 있던 다른 사용자에서 회수. → 한 폰이 환자·이웃 기록에 중복 매핑돼 **`ALERT_SELF`("괜찮으신가요?")와 `NEIGHBOR_ALERT`("출동")가 같은 폰에 이중** 도착하던 문제 해결. (역할 전환/재등록 시 이전 역할 알림이 안 옴)
- **경보음(길게)**: 놓치지 않도록 알림음을 짧은 기본음이 아니라 **약 25초 사이렌**(`res/raw/siren.wav`)으로. 채널 사운드는 생성 시 고정이라 새 채널 id(`emergency_siren`)로 적용 — 기존 설치는 **재설치 or 앱 데이터 삭제** 후 재등록해야 새 채널 생성됨.

## 6. 실패·엣지 대비
| 리스크 | 대비 |
|---|---|
| 아이폰 사이렌 무음 | **역할을 탭**으로 선택(오디오 무장). 무음/방해금지 해제, 볼륨 최대 |
| 갤럭시 푸시 안 옴 | 알림 권한 허용? 콘솔 `push-token 등록` 떴나? Google Play Services 정상? `dev:all`+터널 켜짐? |
| 마이크 거부/미지원 | 자동 **텍스트 입력 폴백** |
| 이웃 미접속(호출 0명) | 갤럭시가 `방림리`로 등록·연결됐는지 확인 |
| 터널 주소 변경 | apk 재빌드 `VITE_BROKER_URL=<새URL> ./scripts/build-apk.sh` (아이폰 Safari는 새 URL만 열면 됨) |
| Claude 키 없음 | AI 추천 폴백 문구(데모 안 깨짐) |
