## 무엇을 담나
- **화면 설계:** 각 화면의 레이아웃·상태(로딩/에러/빈/성공)·인터랙션
- **디자인 토큰:** 색상(고대비)·타이포·간격·반경
- **접근성:** WCAG AA (터치 타겟 ≥ 44px, 명도 대비 ≥ 4.5:1)
- **컴포넌트:** 재사용 UI 규격
