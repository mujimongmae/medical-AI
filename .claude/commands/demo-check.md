---
description: 데모 직전 최종 점검 체크리스트
---

데모 직전 점검을 수행하고 결과를 체크리스트(✅/❌)로 보고해줘:

1. `npm run dev` 로 앱이 정상 기동하는가
2. 핵심 유저 플로우(앱 컨셉의 메인 시나리오)가 end-to-end로 동작하는가
3. 브라우저 콘솔에 에러가 없는가 (chrome-devtools MCP로 확인 가능)
4. 하드코딩된 시크릿 / 실제 개인정보(PII)가 없는가 — grep 스캔
5. (진단성 AI 응답이 있다면) 의료 디스클레이머가 표시되는가
6. (응급 관련 기능이 있다면) 응급 상황에 119/응급실 안내가 우선 노출되는가
7. 모바일 화면(반응형)에서 깨지지 않는가

❌ 항목은 우선순위와 즉시 수정안을 함께 제시할 것.
